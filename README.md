## 📌 Overview

Heart disease remains one of the leading causes of death worldwide. This project leverages Machine Learning to analyze patient health data and provide a prediction regarding the likelihood of heart disease.

The application is built using **Python**, **Scikit-learn**, and **Streamlit**, allowing users to interact with the model through a simple and intuitive web interface.

## 📊 Input Features

The model uses the following patient information:

| Feature                | Description                       |
| ---------------------- | --------------------------------- |
| Age                    | Patient age                       |
| Sex                    | Male / Female                     |
| Chest Pain Type        | ATA, NAP, TA, ASY                 |
| Resting Blood Pressure | Blood pressure at rest            |
| Cholesterol            | Serum cholesterol level           |
| Fasting Blood Sugar    | >120 mg/dl                        |
| Resting ECG            | ECG results                       |
| Maximum Heart Rate     | Achieved heart rate               |
| Exercise Angina        | Exercise-induced angina           |
| Oldpeak                | ST depression value               |
| ST Slope               | Slope of peak exercise ST segment |
